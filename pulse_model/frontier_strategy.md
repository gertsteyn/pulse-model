# Frontier Strategy Toward New Physics

This document records the most promising ways the Pulse Model could be modified, sharpened, or constrained so that it might lead to something genuinely new.

These are not accepted results. They are frontier paths. Each path must preserve the conservative work already completed: known physics first, no hidden preferred frame, no fitted-after-the-fact coefficients, no arbitrary branch labels as physical inputs, and no claim of new physics without a falsifiable observable.

The current status is simple:

- the Pulse Model is a strong conservative reformulation of known clock, phase, gravity, and stress-energy physics
- the model now has useful diagnostics for raw event-graph reconstruction, curvature, scalarization, finite-loop effects, oriented loop phase, spin/full-connection holonomy, branch distinguishability, quantum source-response, and vacuum phase-response
- no current correction is accepted as a new-physics prediction
- the next breakthrough must turn one diagnostic or missing assumption into a real source-response or phase-response law
- the most direct frontier is now the H6 quantum source-response program

## Strategic Shift

The conservative recovery ladder has done its job. More recovery of standard GR or standard QM is not the shortest path to novelty.

The next frontier is not:

- another Schwarzschild, Kerr, de Sitter, or weak-field benchmark
- another Einstein-Hilbert variation proof
- another Regge/action-polishing pass
- another bookkeeping-only H6 summary
- a vacuum/cosmology proposal without a concrete conserved response functional

The next frontier is:

> When matter phase is quantum-superposed, what physical pulse-count geometry does a probe compare against?

The Pulse Model should now force a source-response choice.

Either geometry responds only to density-operator expectation data, in which case H6 probably remains conservative, or some additional physical structure makes branch pulse-count variance real. That additional structure must be local, causal, conservation-respecting, and coefficient-disciplined.

## Project Result Classes

Every new task must classify its result as exactly one of:

- known-physics reformulation
- diagnostic tool
- conditional derivation
- new prediction
- controlled modification
- clean no-go

If a result is only a known-physics reformulation, it must say which later frontier task it supports.

If a result is only a diagnostic tool, it must say which missing assumption it exposes or constrains.

If a result is a controlled modification, it must define all new coefficients, regulators, causal assumptions, and conservation ledgers before external comparison.

A result becomes a new prediction only when it produces a fixed observable distinction from accepted GR+QM baselines.

## Completed First Tweak: Oriented Loop Phase

The highest-leverage first tweak was to make the oriented loop phase itself operational.

The current curvature estimator can recover signed curvature information from loop records, but it does not by itself select a physical geometry phase. A statistical reconstruction loss is naturally squared and sign-blind. The Einstein-Hilbert bridge needs a linear oriented defect.

The tweak was:

> Pulse loop records should carry an additive oriented physical phase increment, not only a timing residual or estimator error.

05S4 completed this path with the verdict **diagnostic tool**. It did not promote Step 5 to a novel geometry-phase bridge. It did define the operational record contract, prove restricted additivity and orientation oddness for corrected phase records, add executable helper checks, and expose the remaining blockers: physical loop-phase readout, phase-to-defect coefficient, scalarization, boundary handling, and source-response.

This remains useful as the contract for future oriented-loop phase claims, but it is no longer the next untested frontier.

## Physical Finite-Loop Scale

Another path is to ask whether finite-loop effects are only estimator resolution, or whether a real physical pulse-resolution scale survives.

The current model treats finite-loop corrections as diagnostics. They are useful for saying when a record is too coarse, but they are not yet physics. To become new physics, the finite-loop scale must be invariant, measurable, and fixed before comparison with observations.

The tweak is:

> A nonzero effective loop scale may be physical rather than merely a reconstruction resolution.

If that is true, finite-loop corrections could map to short-distance gravity deviations, strong-curvature corrections, or gravitational-wave propagation effects.

This path is risky. Without a derived physical scale and source-response map, it collapses into ordinary higher-curvature effective field theory with renamed coefficients. It should be pursued only after the model can say what fixes the scale and why it cannot be refined away.

Finite-loop work should remain paused unless a later H6 source-response law predicts a physical pulse-resolution scale.

## Completed Second Tweak: Spin And Full Connection Holonomy

The second completed tweak extended the pulse-holonomy and phase-response work beyond scalar, electromagnetic, and point-particle matter.

Before 05S5, the H4 matter-side result was accepted with limits for standard scalar, electromagnetic, and point-particle systems. It did not yet cover spinors, spin connection response, nonminimal couplings, or possible torsion-like structure.

The tweak was:

> Pulse phase may couple to the full local connection structure, not only to metric curvature sampled by scalar clock records.

05S5 asked whether spin-dependent phase transport adds an independent observable beyond the metric-only Levi-Civita picture, while staying covariant and avoiding a preferred frame.

The result is:

- standard spinor phase-response reduces cleanly to conservative tetrad and spin-connection bookkeeping
- torsion-free spin and polarization holonomy are representation lifts of H3 frame holonomy
- a bounded Lorentz-connection residual diagnostic is now executable after artifact ledgers
- torsion-like claims remain unsupported without translational closure records
- nonmetricity-like claims remain unsupported without length, angle, or inner-product drift records
- no novel connection-phase response, coefficient rule, or source-response law is accepted

05S5 completed this path with the verdict **diagnostic tool**. It is a major diagnostic level-up, but not a breakthrough into new physics.

Spin/full-connection work should remain paused unless a later source-response law predicts spin-dependent, torsion-like, or connection-level pulse effects with fixed coefficients and clean observables.

## Quantum Source-Response Program

The H6 program is now the main frontier.

H6S1 strengthened H6 from bookkeeping into an executable diagnostic for quantum sources. It compared expectation-sourced weak-field response, branch-specific response, collapse/decoherence placeholders, and a pulse-native response-kernel contract. In the simplest weak-field setup, ensemble means are degenerate. The first possible distinction is distributional or branch-correlated: variance, bimodality, source-probe correlation, clock visibility, or timing of selection relative to a probe record.

The core tweak is:

> Geometry may respond to quantum pulse-history branches through a rule that is neither simple expectation-value sourcing nor an unconstrained collapse postulate.

The H6S1 verdict is **diagnostic tool**. It does not yet supply the rule named by the tweak. It supplies the weak-field source/probe calculations, artifact ledgers, conservation and no-signaling guardrails, and the clean rejection of a toy rule whose probe marginal depends on a remote basis choice without a causal channel.

H6S1's most important lesson is:

> The ensemble mean is not enough.

A later task must decide whether distributional effects are physical or merely artifacts of an arbitrary branch decomposition.

The hard requirements are severe:

- gauge-invariant branch comparison
- local energy and momentum accounting
- no controllable faster-than-light signaling
- a stable branch basis derived from records, environment, or declared preparation, not chosen by convenience
- ensemble-decomposition invariance unless a local record physically selects the ensemble
- a prediction beyond the ensemble mean
- fixed coefficients before external comparison

This remains the closest path to an experimentally distinguishable beyond-GR/QM signal if a later task can supply the missing response law. Candidate target classes remain source superpositions, quantum clocks, branch-correlated probe timing, visibility-loss probes, stochastic pulse-noise probes, or gravitationally mediated entanglement style tests.

## H6S2: Ensemble-Invariant Causal Pulse-Response Kernel

H6S2 is now complete as the immediate successor to H6S1.

Its purpose is to prevent fake novelty. The canonical result is [H6S2 Causal Pulse-Response Kernel](./appendix/h6/causal_pulse_response_kernel.md), with executable helpers in `src/pulse_model/causal_pulse_response.py` and focused tests in `tests/test_causal_pulse_response.py`.

H6S1 showed that a branch-mixture response can differ from an expectation-sourced response in variance, bimodality, source-probe correlation, and visibility. But a branch mixture is not automatically physical. A quantum density operator can be decomposed into many different ensembles.

For example:

$$
\rho_S = \sum_i p_i |\psi_i\rangle\langle\psi_i| = \sum_j q_j |\phi_j\rangle\langle\phi_j|
$$

If a response rule depends on the chosen ensemble decomposition rather than on the local density operator or on actual local records, then the probe marginal can become basis-dependent.

The H6S2 guardrail is:

> A valid pulse-response kernel must be invariant under different ensemble decompositions of the same local density operator unless the difference is tied to an actual local record inside the probe's causal past or to a declared causal channel.

In response-kernel form:

$$
\mathcal R(\{p_i, |\psi_i\rangle\})
=
\mathcal R(\{q_j, |\phi_j\rangle\})
$$

whenever both ensembles represent the same local state and no physical record selects one decomposition.

H6S2 documents and implements four families.

### H6S2-A: Density-Operator Expectation Response

This is the conservative baseline.

Given a local source state $\rho$, a weak-field potential operator $\hat\Phi$, probe-clock frequency $f_C$, and duration $T$:

$$
N_{\mathrm{exp}} = f_C T\left(1 + \frac{\operatorname{Tr}(\rho\hat\Phi)}{c^2}\right)
$$

This response depends on $\rho$, not on an arbitrary ensemble decomposition.

Expected result class: **known-physics reformulation**.

### H6S2-B: Pointer-Record Branch Response

A branch response can be admissible only if the branch basis is physically selected by local records, environment, or a declared preparation protocol.

A valid pointer-record response has the form:

$$
P(N_C) = \sum_a P(a\mid R_{\mathrm{local}})\,\delta(N_C-N_a)
$$

where $R_{\mathrm{local}}$ is a pointer or environment record available in the probe's causal past, or the distribution is only compared later in a shared causal future.

Expected result class: **diagnostic tool**.

This does not discover a new law. It states when branch-conditioned variance is a physical target rather than an artifact.

### H6S2-C: Invalid Ensemble-Dependent Response

A response of the form:

$$
P(N_C) = \sum_i p_i\delta(N_C-N_i)
$$

is invalid if the $i$ labels are merely an externally chosen ensemble decomposition of the same $\rho$.

H6S2 rejects such a rule when changing the ensemble changes the probe marginal without a local record or causal channel.

Expected result class: **clean no-go** subtheorem inside a **diagnostic tool**.

### H6S2-D: Pulse-Native Candidate Contract

H6S2 does not invent the final law.

It should define the admissible contract:

$$
\mathcal R:
(\rho_{\mathrm{local}}, R_{\mathrm{pulse}}, R_{\mathrm{env}}, L_{\mathrm{cons}}, D_{\mathrm{causal}}, L_{\mathrm{coeff}})
\rightarrow
P(N_C)
$$

The response must provide:

- normalized output distribution
- ensemble-decomposition invariance
- causal support only
- no remote-basis marginal dependence
- conservation classification
- gauge and branch-matching declaration
- coefficient and regulator provenance
- artifact ledger
- pointer-record conditioning only when records exist

Expected result class: **diagnostic tool**.

### H6S2 Required Theorem

H6S2 proves:

> If two ensemble decompositions represent the same local density matrix and a response rule assigns different probe marginals solely because one decomposition rather than another was chosen, then the response rule is not an admissible spacelike source-response law.

Proof structure:

1. Remote steering can prepare different ensemble decompositions of the same reduced local state without changing the local density operator.
2. If the gravitational or pulse response depends on the ensemble decomposition rather than on $\rho$ or local records, the remote party can change the local probe marginal by choosing the steering basis.
3. That violates the H6S1 no-signaling guardrail.
4. Therefore the response must be density-operator based, local-record conditioned, or supplied with a causal channel/selection law.

This theorem does not prove gravity is quantum, does not rule out all hybrid classical-quantum gravity models, and does not discover the Pulse Model source-response law. It removes an inadmissible shortcut.

### H6S2 Impact On H6S1

H6S1 says:

> Mean is degenerate; variance, bimodality, correlation, and visibility may discriminate response families.

H6S2 adds:

> Distributional channels are admissible only when the branch basis is supported by local records, a causal selection rule, or comparison in a shared causal future.

Without this, branch-mixture variance can be an ensemble-decomposition artifact.

## H6S3: No Free Branch Variance Theorem

H6S3 should come after H6S2.

H6S2 blocks arbitrary ensemble-dependent response rules. H6S3 should turn that block into the central source-response trilemma.

The theorem should say:

> In a source-superposition experiment, a probe-clock pulse distribution cannot gain physical branch variance merely because we choose a branch decomposition. Physical variance beyond ordinary measurement noise must come from one of three things: local pointer/collapse records, objectively stochastic classical pulse geometry, or non-classical pulse geometry.

Equivalently:

$$
\rho_S
=
\sum_i p_i|\psi_i\rangle\langle\psi_i|
=
\sum_j q_j|\phi_j\rangle\langle\phi_j|
$$

If the probe has access only to $\rho_S$ and no physical branch record, then an admissible response must satisfy:

$$
\mathcal R(\{p_i,\psi_i\})
=
\mathcal R(\{q_j,\phi_j\})
$$

Therefore branch-dependent pulse variance is not available for free.

The resulting trilemma is:

| Route | Meaning | Status |
|---|---|---|
| Density-only response | Geometry responds to $\rho$ or $\langle T_{\mu\nu}\rangle$ | Conservative baseline |
| Pointer-record response | Branches become physical only when records exist | Diagnostic / collapse-adjacent |
| Stochastic classical response | Geometry remains classical but gains objective noise or irreversibility | Potential controlled modification |
| Quantum geometry response | Geometry or mediator carries branch information non-classically | Quantum-gravity route |

H6S3's likely result class is **diagnostic tool** with a **clean no-go** subtheorem.

It becomes useful because it forces the next task to choose an allowed escape route instead of treating branch labels as physical by notation.

## H6S4: Choose An Escape Route

After H6S3, the model must stop adding guardrails and choose one admissible route to explore.

The choice should be made explicitly. Each route has a different route to novelty and a different failure mode.

### H6S4-D: Density-Only Response Baseline

This route says:

$$
N_C = f_C T\left(1 + \frac{\operatorname{Tr}(\rho\hat\Phi)}{c^2}\right)
$$

It is ensemble-invariant, causal in the weak-field operational sense, and conservative.

Strength:

- clean baseline
- no arbitrary branch basis
- useful comparison model

Weakness:

- likely no new H6 distributional signal
- may inherit known semiclassical-gravity problems
- not enough to bridge GR and QM by itself

Expected result class: **known-physics reformulation** or **diagnostic tool**.

This route should not be the main novelty route, but it should remain the required baseline.

### H6S4-P: Pointer-Record Or Collapse-Selection Pulse Law

This route says:

> Branches become physical only when local records actualize them or when a declared selection law chooses them.

Then branch-conditioned probe shifts are allowed only after the record is inside the probe's causal past or when correlations are compared later in a shared causal future.

Required law shape:

$$
\mathcal R_P:
(\rho, R_{\mathrm{pointer}}, D_{\mathrm{causal}}, L_{\mathrm{cons}})
\rightarrow P(N_C)
$$

This route must answer:

- what counts as a physical pointer pulse record
- when the record becomes available to the probe
- how branch selection avoids remote-basis signaling
- where energy-momentum accounting goes during selection
- whether the selection timing changes clock visibility or probe variance

Strength:

- naturally respects H6S2 if records are local
- can produce branch-conditioned observables
- connects to decoherence and collapse-style models

Weakness:

- conservation is difficult
- selection timing can be ambiguous
- may collapse into ordinary decoherence bookkeeping without a new law

Expected result class: **controlled modification** only if it defines a concrete selection rule and conservation ledger. Otherwise **diagnostic tool**.

### H6S4-C: Minimal Stochastic Classical Pulse Geometry

This is the most direct candidate route to new physical law while keeping geometry classical.

The idea:

> The pulse-count metric remains classical but has objective stochastic fluctuations tied to quantum source uncertainty.

The probe does not see arbitrary branch variance. It sees lawful, causal pulse-count noise:

$$
\mathcal R_C:
(\rho_{\mathrm{local}}, R_{\mathrm{pulse}}, D_{\mathrm{causal}}, L_{\mathrm{cons}}, L_{\mathrm{coeff}})
\rightarrow P(N_C)
$$

A minimal stochastic pulse-geometry law must specify:

- the stochastic variable or noise process
- whether noise couples to $T_{\mu\nu}$, phase density, clock records, or curvature records
- causal support of the noise correlations
- conservation status branchwise, in expectation, or environment-assisted
- coefficient origin
- GR/QM recovery limits
- predicted probe-clock variance, covariance, visibility loss, or timing jitter

The central observable would be an excess pulse-count distribution:

$$
\operatorname{Var}(N_C)
=
\operatorname{Var}_{\mathrm{known}}(N_C)
+
\operatorname{Var}_{\mathrm{pulse\ noise}}(N_C)
$$

where $\operatorname{Var}_{\mathrm{pulse\ noise}}$ is fixed before external comparison.

Strength:

- can produce a real distributional discriminator
- may remain classical in geometry while modifying source-response
- naturally fits the Pulse Model language of pulse-count records and stochastic comparisons

Weakness:

- high risk of becoming already-known stochastic semiclassical gravity with renamed variables
- conservation and no-signaling constraints are severe
- coefficients must be derived or fixed before tests

Expected result class: **controlled modification**. It becomes **new prediction** only if it fixes a nonzero observable effect not equivalent to known models.

This is the recommended first escape route after H6S3.

### H6S4-Q: Quantum Pulse Geometry

This route says:

> If pulse-count geometry can carry branch information coherently, then geometry or the mediator itself has non-classical degrees of freedom.

The response is not a classical distribution over pulse counts alone. It may require a quantum state for the mediator or geometry record:

$$
\rho_{SCP}
\quad\mathrm{or}\quad
|S\rangle|C\rangle|G\rangle
$$

where $S$ is the source, $C$ is the clock/probe, and $G$ is the geometry or mediator sector.

This route must answer:

- what the geometry or mediator Hilbert space is
- how pulse records are read without introducing a preferred frame
- how the classical metric emerges in the conservative limit
- whether mediator records can entangle source and probe
- how the model differs from ordinary linearized quantum gravity
- what clock-network observable is uniquely pulse-native

Strength:

- directly targets the GR/QM bridge
- can make branch correlation and visibility effects physical without arbitrary ensemble labels
- connects naturally to quantum-clock and gravitationally mediated entanglement tests

Weakness:

- easy to become standard quantum gravity in new language
- requires much more structure than H6S1/H6S2
- conservation, gauge, and measurement rules become harder

Expected result class: **controlled modification** or **conditional derivation**. It becomes **new prediction** only if it produces a fixed observable distinction from existing quantum-gravity baselines.

This is the second recommended escape route after H6S3.

## H6S5: Observable Map

H6S5 should take the chosen H6S4 route and produce a concrete observable map.

It must compare against at least these baselines:

- standard quantum matter on a fixed weak-field background
- expectation-sourced semiclassical response
- pointer-record/decoherence bookkeeping
- stochastic classical response if H6S4-Q is chosen
- quantum mediator response if H6S4-C is chosen

It must not compare only ensemble means.

Allowed observable classes include:

- excess pulse-count variance
- bimodality or branch-separation score
- source-probe correlation after shared-future comparison
- conditional probe shift after source readout
- internal clock visibility loss
- timing of branch selection relative to probe record
- clock-network covariance structure
- causal-domain dependence of correlations

H6S5 should output a table:

| Observable | Standard baseline | Candidate law | Difference | Coefficient status | Causal status | Conservation status |
|---|---|---|---|---|---|---|

A claimed observable is not allowed unless its coefficient status is one of:

- derived
- calibrated-before-test
- fixed-by-known-limit

A result with `exploratory-only` coefficient status remains a diagnostic, not a prediction.

## H6S6: Coefficient And Falsifiability Ledger

H6S6 should prevent fitted-after-the-fact physics.

Every new law or correction must declare:

- all free coefficients
- dimensions and normalization
- whether each coefficient is derived, fixed by a known limit, calibrated before the test, or exploratory-only
- regulator dependence
- artifact subtraction method
- external data not used during coefficient choice
- falsification threshold

A candidate law cannot be promoted to **new prediction** unless every coefficient that affects the target observable is derived, fixed by a known limit, or calibrated before comparison.

## H6S7: External Comparison

Only after H6S4-H6S6 should the project compare to external experiments or bounds.

Possible target families:

- atomic clock networks
- clock interferometry
- source-superposition gravity probes
- gravitationally mediated entanglement style experiments
- optomechanical mass-superposition probes
- precision redshift and visibility-loss tests

External comparison should be used only after the model has a fixed observable.

Do not use external bounds to select the law.

## Distance From Novelty

The project is close to theoretical usefulness but not yet close to accepted new physics.

The rough ladder is:

| Level | Meaning | Current location |
|---|---|---|
| 0 | Conservative reformulation | Achieved in H1-H5 and H7 |
| 1 | Diagnostic framework | Achieved in H2S1, H3/05S diagnostics, 05S5, H6S1 |
| 2 | Clean theorem/no-go | H6S2 achieved a no-go subtheorem; H6S3 target remains |
| 3 | Controlled modification | H6S4 target |
| 4 | Fixed new prediction | H6S5-H6S6 target |
| 5 | External empirical support | After H6S7 |

The model is probably one strong theorem away from a useful theoretical contribution:

> No branch pulse-count variance without physical records, stochastic pulse geometry, or non-classical pulse geometry.

The model is several hard gates away from new physics:

1. ensemble-invariance guardrail
2. no-free-branch-variance theorem
3. one admissible response law
4. fixed observable consequences
5. coefficient discipline
6. comparison against existing semiclassical, stochastic, collapse, and quantum-mediator frameworks
7. external test

New physics begins at H6S4/H6S5 only if the chosen law is specific and not equivalent to existing frameworks.

Accepted new physics requires H6S7 or later.

## Recommended Order

The recommended order is now:

1. Completed: **H6S2: Ensemble-invariant causal pulse-response kernel**
2. **H6S3: No free branch variance theorem**
3. **H6S4-C: Minimal stochastic classical pulse-geometry candidate**
4. **H6S4-Q: Quantum pulse-geometry comparison candidate**
5. **H6S5: Observable map**
6. **H6S6: Coefficient and falsifiability ledger**
7. **H6S7: External comparison**
8. Physical finite-loop scale only if H6 predicts an invariant scale
9. Vacuum residual law only if a concrete conserved functional is proposed

The reason is practical. The oriented loop phase path has already been sharpened into a diagnostic contract, spin/full-connection holonomy has been sharpened into a conservative recovery plus bounded diagnostic, and H6S1 has sharpened quantum source-response into executable weak-field gates. All three point at the same missing ingredient: a causal, conservation-respecting source-response law.

Finite-loop and vacuum routes should wait unless the next proposal can distinguish physical response from estimator artifacts and fitted parameters.

## Novelty Gate

A frontier path should be considered promising only if it passes this gate:

- it changes a missing assumption into a measurable, derivable, or falsifiable statement
- it preserves the accepted known-physics recovery ladder
- it does not introduce an unobserved preferred frame unless it predicts bounded Lorentz violation
- it supplies a source-response or phase-response map before comparing with external bounds
- it fixes or measures any new coefficient before using observational constraints
- it is ensemble-decomposition invariant unless local records or causal selection justify branch conditioning
- it has a conservation ledger for energy and momentum accounting
- it has a causal-domain ledger for every branch, record, and response input
- it produces either a new derivation, a new prediction, a stronger diagnostic, or a clean no-go

If a path cannot pass this gate, it may still be useful as bookkeeping, but it should not be treated as a route to new physics.

## Work To Avoid Next

Do not do these next:

- more Schwarzschild/Kerr/de Sitter benchmarks unless required by a target H6 observable
- more Einstein-Hilbert variation proofs
- more Regge action polishing without deriving locality, scalarization, refinement, and coefficient assumptions
- more H7 vacuum text without a concrete conserved functional
- new torsion, nonmetricity, or spin-response claims without fixed observables
- importing external bounds before the model has fixed coefficients
- treating branch labels as physical without local pointer records or causal selection
- treating branch-mixture variance as new physics without H6S2-H6S3 guardrails
- creating a quantum metric Hilbert space without a scoped H6S4-Q contract

## Current Recommendation

H6S2 is complete; the next frontier should build on it, not repeat another H6 summary.

H6S2 establishes ensemble-decomposition invariance for admissible pulse-response kernels. It shows that branch-mixture response is not an admissible spacelike law merely because a branch decomposition can be written down. Branch-conditioned variance becomes a valid target only when the branch basis is supported by local pointer records, a causal selection rule, or comparison in a shared causal future.

After H6S2, H6S3 should prove the no-free-branch-variance theorem and force the model into an explicit trilemma. After that, the first serious novelty attempt should be H6S4-C: a minimal stochastic classical pulse-geometry kernel. H6S4-Q should be developed in parallel only as the quantum-geometry comparator.

The pulse-native law remains open.

The seam to pull is:

> What physical thing makes branch pulse-count variance real?

If the answer is nothing, the Pulse Model remains a conservative diagnostic framework.

If the answer is stochastic pulse geometry or non-classical pulse geometry, and it produces a fixed, causal, conservation-respecting observable, that is the first real step toward new physics.
